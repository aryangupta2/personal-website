# Personal Website

A simple personal portfolio site built with Astro and Tailwind CSS.

## Run Locally

Clone repo:

```sh
git clone https://github.com/aryangupta2/personal-website.git
```

```sh
cd personal-website
```

From the project root:

```sh
bun install
bun dev
```

## Useful Commands

- `bun dev` - start the local development server
- `bun build` - build the production site
- `bun preview` - preview the production build locally
